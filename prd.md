# PRD: GARAGE 70/80 — информационный сайт о ретро-автомобилях

## 1. Обзор продукта

**Название:** GARAGE 70/80
**Тип продукта:** Информационный веб-сайт (одностраничное приложение)
**Тема:** Классические легковые автомобили США, Японии и Германии 1970–1989 годов

Сайт позволяет пользователю каскадно выбрать страну → класс автомобиля → конкретную модель и получить карточку с техническими характеристиками, кратким описанием и иллюстрацией автомобиля.

## 2. Цели продукта

| Цель | Метрика успеха |
|---|---|
| Дать пользователю быстрый и понятный способ находить ретро-автомобили по стране и классу | Выбор модели занимает не более 3 действий (страна → класс → модель) |
| Представить достоверные технические характеристики моделей | Каждая карточка содержит полный набор из 10 характеристик |
| Обеспечить визуальное разделение по странам-производителям | Каждая страна имеет собственный цветовой бейдж |
| Дать целостный, тематически выдержанный визуальный стиль эпохи 70–80-х | Единая дизайн-система (типографика, палитра, иллюстрации) |

## 3. Целевая аудитория

- Любители классических автомобилей и автоистории
- Пользователи, сравнивающие технические параметры машин разных стран/эпох
- Посетители, которым нужен быстрый справочник по моделям 70–80-х годов

## 4. Пользовательские сценарии (User Stories)

1. **Как посетитель**, я хочу выбрать страну производителя, чтобы увидеть только автомобили из США, Японии или Германии.
2. **Как посетитель**, я хочу выбрать класс автомобиля (маслкар, спорткар, купе, седан, компакт, люкс, внедорожник), чтобы сузить список моделей.
3. **Как посетитель**, я хочу выбрать конкретную модель из меню, чтобы увидеть её полную карточку характеристик.
4. **Как посетитель**, я хочу кликнуть на карточку в общей сетке, чтобы сразу открыть подробную информацию, не пользуясь меню.
5. **Как посетитель**, я хочу видеть иллюстрацию автомобиля и его ключевые параметры (мощность, разгон, макс. скорость) в наглядном виде (спидометр, шкала мощности).

## 5. Функциональные требования

### 5.1 Навигация и фильтрация
- Три выпадающих меню: **Страна**, **Класс**, **Модель**.
- Меню «Класс» динамически перестраивается в зависимости от выбранной страны (показываются только классы, представленные в этой стране).
- Меню «Модель» динамически перестраивается в зависимости от выбранных страны и класса.
- Значения по умолчанию — «Все страны» / «Все классы» / без выбранной модели.
- Счётчик найденных моделей отображается под меню.

### 5.2 Сетка автомобилей
- Отображает все модели, соответствующие текущему фильтру страны/класса.
- Каждая карточка содержит: иллюстрацию, бейдж страны (с флагом), бейдж класса, название модели, год, мощность.
- Клик по карточке открывает детальную карточку модели и синхронизирует значения меню.
- Если по фильтру нет ни одной модели — показывается пустое состояние с поясняющим текстом.

### 5.3 Детальная карточка модели
Отображается при выборе модели (через меню или клик по карточке) и содержит:
- Иллюстрацию автомобиля (силуэт, цвет кузова индивидуален для модели)
- Бейджи: страна, класс, годы выпуска
- Название и модельный год
- Краткое описание (2–3 предложения об истории/особенностях модели)
- Спидометр (SVG-шкала) с анимированной стрелкой, отражающей максимальную скорость
- Шкалу мощности двигателя (прогресс-бар)
- Таблицу технических характеристик:
  - Двигатель
  - Мощность
  - Крутящий момент
  - Разгон 0–100 км/ч
  - Максимальная скорость
  - Снаряжённая масса
  - Привод (передний/задний/полный)
  - Годы выпуска
  - Класс
  - Ориентировочная цена на момент выпуска

### 5.4 Контент (данные по автомобилям)
База данных сайта содержит **24 модели** — по 8 на страну:

**США:** Ford Mustang Mach 1 (1973), Chevrolet Camaro Z28 (1978), Pontiac Firebird Trans Am (1977), Dodge Charger SE (1975), Chevrolet Corvette C3 (1980), Cadillac Eldorado (1976), Lincoln Continental Mark V (1977), Jeep CJ-7 (1979).

**Япония:** Toyota Celica GT (1978), Datsun 280Z / Fairlady Z (1978), Mazda RX-7 SA22C (1979), Honda Civic CVCC (1979), Toyota Corolla E30 (1979), Datsun 510 (1973), Mitsubishi Galant Lambda (1980), Subaru Leone 4WD Estate (1979).

**Германия:** BMW 3.0 CSL (1972), BMW M1 (1978), Mercedes-Benz 450 SL / R107 (1973), Porsche 911 Turbo / 930 (1975), Volkswagen Golf GTI Mk1 (1976), Audi Quattro (1980), Mercedes-Benz 280E / W123 (1976), Opel Manta GT/E (1975).

Классы, представленные в каталоге: Маслкар, Спорткар, Купе, Седан, Компакт, Люкс, Внедорожник.

## 6. Нефункциональные требования

- **Адаптивность:** корректное отображение на десктопе и мобильных устройствах (брейкпоинты для сетки, меню и детальной карточки).
- **Производительность:** статичный HTML/CSS/JS без внешних зависимостей (кроме шрифтов Google Fonts), мгновенная фильтрация без перезагрузки страницы.
- **Доступность:** видимый фокус для интерактивных элементов, уважение настройки `prefers-reduced-motion`.
- **Локализация:** интерфейс и контент на русском языке.

## 7. Дизайн-система

| Фирменный элемент (signature) | Анимированный SVG-спидометр в детальной карточке модели |

## 8. Технические ограничения / известные компромиссы

- Фотографии моделей взяты с Wikimedia Commons под свободными лицензиями (Public Domain, CC BY / CC BY-SA) — использование в проекте правомерно при условии указания автора и лицензии, которое отображается под каждой фотографией.
- Для части моделей (Dodge Charger SE, Mitsubishi Galant Lambda) точных фотографий указанной трим-версии/экспортного бейджа в свободном доступе не нашлось — использованы фото того же поколения/платформы, что указано в подписи к фото.
- Указанные технические характеристики и цены являются ориентировочными, основаны на общедоступных исторических данных по моделям и могут отличаться в зависимости от рынка/комплектации.

## 9. Вне рамок текущей версии (Out of scope)

- Поиск по свободному тексту
- Сравнение нескольких моделей "бок о бок"
- Пользовательские аккаунты, избранное, отзывы
- Мультиязычность интерфейса
- Реальные фотографии/галереи изображений
- Серверная часть / база данных (данные хранятся статично во фронтенд-коде)

## 10. Возможные направления развития

- Добавление реальных лицензированных фотографий и мультиизображений (интерьер, экстерьер, детали)
- Расширение каталога новыми странами/моделями
- Функция сравнения двух моделей в одной таблице
- Сохранение "избранных" моделей в локальном хранилище браузера
- Раздел с историческим контекстом эпохи (нефтяной кризис, нормы выбросов и т.д.)

## 11. Технологический стек

- HTML5 / CSS3 (кастомные CSS-переменные, без фреймворков)
- Vanilla JavaScript (без сборщиков и зависимостей)
- Google Fonts (Oswald, Space Grotesk, JetBrains Mono)
- Все иллюстрации — inline SVG, генерируемые на клиенте
