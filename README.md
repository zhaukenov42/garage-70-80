# GARAGE 70/80

Информационный сайт о классических автомобилях США, Японии и Германии 1970–1989 годов. Каскадный выбор страны → класса → модели, сетка карточек и детальная карточка модели с анимированным SVG-спидометром и шкалой мощности.

Реализовано на чистых HTML/CSS/JS без сборщиков и зависимостей (см. [prd.md](prd.md)).

## Запуск

Откройте `index.html` в браузере или поднимите любой статический сервер, например:

```
npx serve .
```
